# EXEDUM dashboard
simple dashboard for interacting with the exedum protocol.

## Prerequisites
- install [Node.js](https://nodejs.org/en/download/).

## Install & run locally
```shell
git clone git@github.com:exedum/exedum-interface.git
cd exedum-interface
npm install

npm start
```
